# Multi-Agent 私人任务助理

## 目标

围绕个人资料、学习资料和研究任务，提供一个可分解任务、可调用工具、可审查结果、可沉淀长期记忆的多 Agent 助理系统。

## 核心叙事

这个项目的产品定位不是“通用聊天机器人”，而是“资料驱动的任务助理”：

- 用户带着资料来
- 系统先查本地资料和长期记忆
- Master 拆任务并派发 Worker
- Checker 区分真正阻塞的缺口和可接受的缺口
- 高价值结果沉淀进长期记忆

项目对外展示时，重点不是回答了什么，而是：

- 为什么这样拆任务
- 为什么继续补救或停止
- 结果依据来自哪里
- 哪些结论被写入了长期记忆

## MVP

- Master 接收任务并拆分子任务
- 通用 Worker 自主调用 RAG / Web Search / MCP / Export / Memory 工具
- Final Checker 评估任务完成度与证据充分性
- 短期记忆管理上下文，长期记忆沉淀高价值结果
- FastAPI 提供运行时与记忆接口
- 前端展示基础运行轨迹与记忆命中/写入

## 典型使用场景

- 基于课程资料生成复习提纲
- 检查资料来源是否完整，必要时联网补充
- 对技术文档做结构化总结并沉淀长期记忆
- 结合历史研究结果快速续写同主题任务

## 后续增强

- 并行 worker
- 更细粒度记忆评分
- 更强近似去重与记忆合并
- 用户偏好自动学习
- 更完整任务面板与观测能力
